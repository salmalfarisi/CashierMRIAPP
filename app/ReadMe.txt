Tools : 
Front-end : Bootstrap
Back-end : CodeIgniter 3
Database : MySQL

Tata cara penggunaan : 
1. buka file database.php di application/config/database.php
2. pastikan data yang ada di dalam database dengan data yang ada di dalam codeigniter sama
3. lakukan import secara manual (tidak menggunakan command prompt sama sekali)
4. project siap dijalankan
